Express + Prisma + TypeScript Documentation

Welcome to the Express API documentation!

Setup Instructions

  1. Clone this repository:
    2. git clone <repository-url>
  2. Navigate to the project directory:
    3. cd express-typescript-prisma
  3. Install dependencies:
    4. npm install
  4. Set up the database connection in the .env file:
    5. DATABASE_URL="file:./dev.db"
PORT=3000
  5. Initialize the database with Prisma:
    6. npx prisma migrate dev --name init
  6. Start the development server:
    7. npm run dev

API Endpoints

1. Get All Users

Retrieve all users.

GET /users

2. Create a User

Create a new user.

POST /users

Body:

{
  "name": "John Doe",
  "email": "john.doe@example.com"
}

Notes

  • Make sure you have Node.js and npm installed.
  • Do not share the .env file publicly.
